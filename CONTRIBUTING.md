# Como Contribuir para Este Projeto
Olá! Agradecemos o seu interesse em contribuir para o nosso projeto de chatbot. Sua colaboração é muito bem-vinda e essencial para o crescimento e melhoria do projeto.
---

### **Reportando Bugs**
Se você encontrar um bug ou um problema no código, por favor, abra uma `issue` no GitHub.
* Inclua uma descrição clara e detalhada do problema.
* Explique os passos para reproduzir o bug.
* Se possível, inclua capturas de tela ou mensagens de erro relevantes.
---

### **Sugerindo Novas Funcionalidades**
Queremos ouvir suas ideias! Se você tem uma sugestão para uma nova funcionalidade, abra uma `issue` para discutir a sua proposta.
* Descreva a funcionalidade que você sugere.
* Explique como ela poderia beneficiar o projeto.
---

### **Enviando Mudanças (Pull Requests)**
Se você corrigiu um bug ou implementou uma nova funcionalidade, você pode enviar um `Pull Request` (PR).
* Faça um `fork` do repositório.
* Crie uma nova `branch` para a sua alteração.
* Faça o `commit` das suas alterações com uma mensagem clara e descritiva.
* Envie o seu `Pull Request` para a `branch main` do nosso repositório.
Se você contribuiu junto com um colega, por favor, use a mensagem de commit com co-autoria:
`Co-authored-by: `
---

Agradecemos a sua colaboração!
